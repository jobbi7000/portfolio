# 포트폴리오 웹사이트

## 소개
개발자 길잡이의 포트폴리오 웹사이트입니다. Next.js와 TypeScript를 사용하여 제작되었습니다.

## 기술 스택
- **Frontend**: React, Next.js, TypeScript
- **Backend**: Node.js, Express, MongoDB
- **Tools**: Git, Docker, AWS
- **Styling**: Tailwind CSS

## 주요 기능
- 📱 반응형 디자인
- 🔒 HTTPS 지원
- 🎨 모던한 UI/UX
- 📞 연락처 정보
- 🔗 소셜 미디어 링크

## 실행 방법

### 개발 서버 실행
```bash
npm run dev
```

### HTTPS 개발 서버 실행
```bash
npm run dev
# 자동으로 HTTPS로 실행됩니다
```

### 프로덕션 빌드
```bash
npm run build
npm start
```
